


Customised Key Mappings
=======================

* `COMAND` -> `Alt`
* `Option` -> `Windows`
* `Control-Delete` -> `Forward Delete`
* `Control-Down` -> `Page Down`
* `Control-Up` -> `Page Up`

Windows specific
----------------

* `Option-Space` -> `Space`
* `Shift-Option-Space` -> `Space`
* `Shift-Control-V` -> `Shift Insert`
* `Control-A` -> `Home`
* `Control-Shift-A` -> `Shift-Home`
* `Control-E` -> `End`
* `Control-Shift-E` -> `Shift-End`
